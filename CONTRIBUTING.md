# Contributing

## Commit conventions

Do not add AI-assistance attribution of any kind to commits or code in this
repo — no `Co-Authored-By: Claude ...` (or any other AI tool) trailers, no
"Generated with ..." footers, and no comments/docstrings/docs crediting an AI
assistant as the author or source. `user.name`/`user.email` on commits must
stay the human author's own identity. Project history should read as
entirely human-authored.

This applies regardless of what tooling was used to help write the code.
